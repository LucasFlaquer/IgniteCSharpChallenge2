# Ignite Challenge

Welcome to the Ignite Challenge, a practical test for a bookstore management system developed in C# .NET. This project aims to demonstrate skills in application development, code organization, and efficient database management, creating an intuitive interface for users.

## Introduction

The Ignite Challenge is designed to simulate the operations of a bookstore. It encompasses the management of books, authors, and customer transactions, offering functionalities for adding, updating, and deleting records while ensuring data integrity and efficient retrieval. 

## Features

- **Book Management**: Add, update, delete, and view books in the inventory.
- **Author Management**: Create and manage author profiles linked to their respective books.
- **Customer Transactions**: Record customer purchases and manage transactions.
- **Search Functionality**: Easily search for books or authors using various filters.
- **Data Persistence**: Store all data using a relational database.

## Technologies Used

- **Programming Language**: C#
- **Framework**: .NET
- **Database**: SQL Server (or any other preferred relational database)
- **Development Environment**: Visual Studio / Visual Studio Code

## Installation

To set up the project locally, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/ignite-challenge.git
   ```

2. Navigate to the project directory:
   ```bash
   cd ignite-challenge
   ```

3. Restore the dependencies:
   ```bash
   dotnet restore
   ```

4. Run the application:
   ```bash
   dotnet run
   ```

## Usage

After running the application, the bookstore management system will be accessible. You can start by adding new books and authors, and then proceed to manage customer transactions. The user interface allows you to navigate through different functionalities seamlessly.
